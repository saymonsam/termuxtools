📡
Nmap
Network discovery & port scanning tool
নেটওয়ার্ক স্ক্যান ও পোর্ট খোঁজার টুল
NETWORK
INSTALL
ইন্সটল
pkg install nmap
COMMANDS
কমান্ড
nmap 192.168.1.1
Basic scan — checks open ports of a host
বেসিক স্ক্যান — হোস্টের খোলা পোর্ট দেখায়
nmap -sV -O 192.168.1.0/24
Detect services & OS version on entire subnet
পুরো সাবনেটে সার্ভিস ও OS ভার্শন ডিটেক্ট করে
nmap -A -T4 target.com
Aggressive scan with OS detection, scripts & traceroute
অ্যাগ্রেসিভ স্ক্যান — OS, স্ক্রিপ্ট ও ট্রেসরুট সহ
🔐
Hydra
Fast network login brute-force tool
নেটওয়ার্ক লগিন ব্রুটফোর্স টুল
SECURITY
INSTALL
ইন্সটল
pkg install hydra
COMMANDS
কমান্ড
hydra -l admin -P pass.txt ssh://192.168.1.1
SSH brute-force with username 'admin' and wordlist
SSH ব্রুটফোর্স — username 'admin' ও পাসওয়ার্ড লিস্ট দিয়ে
hydra -L users.txt -P pass.txt ftp://target
FTP brute-force with user and password lists
FTP ব্রুটফোর্স — ইউজার ও পাসওয়ার্ড লিস্ট দিয়ে
🌐
SQLmap
Automatic SQL injection detection & exploitation
SQL ইনজেকশন অটোমেটিক ডিটেক্ট ও এক্সপ্লয়েট করার টুল
SECURITY
INSTALL
ইন্সটল
pkg install sqlmap
COMMANDS
কমান্ড
sqlmap -u "http://site.com/page?id=1" --dbs
Test URL for SQL injection and list databases
URL-এ SQL injection টেস্ট করে ডেটাবেস লিস্ট দেখায়
sqlmap -u "url" -D dbname --tables
List tables inside a specific database
নির্দিষ্ট ডেটাবেসের টেবিল লিস্ট দেখায়
📦
pkg / apt
Termux package manager — install, update, remove tools
Termux প্যাকেজ ম্যানেজার — টুলস ইন্সটল, আপডেট, রিমুভ করুন
PACKAGE
pkg update && pkg upgrade
Update all packages to latest version
সব প্যাকেজ লেটেস্ট ভার্শনে আপডেট করুন
pkg install tool-name
Install any package/tool
যেকোনো প্যাকেজ/টুল ইন্সটল করুন
pkg remove tool-name
Uninstall a package
প্যাকেজ আনইন্সটল করুন
pkg search keyword
Search for available packages
উপলব্ধ প্যাকেজ খুঁজুন
📁
File Commands
Navigate, create, copy, move and manage files
ফাইল নেভিগেট, তৈরি, কপি, মুভ ও ম্যানেজ করুন
FILE
ls -la
List all files (including hidden) with details
সব ফাইল (লুকানো সহ) বিস্তারিত দেখান
mkdir folder_name
Create a new directory/folder
নতুন ডিরেক্টরি/ফোল্ডার তৈরি করুন
cp source dest
Copy a file from source to destination
ফাইল কপি করুন সোর্স থেকে গন্তব্যে
rm -rf folder/
⚠️ Force delete a folder and all its contents
⚠️ ফোল্ডার ও সব কন্টেন্ট জোর করে ডিলিট করুন
⚙️
System Info
Monitor CPU, memory, processes and system status
CPU, মেমরি, প্রসেস ও সিস্টেম স্ট্যাটাস মনিটর করুন
SYSTEM
top
Show running processes with CPU & memory usage
চলমান প্রসেস ও CPU/মেমরি ব্যবহার দেখুন
df -h
Show disk space in human-readable format
ডিস্ক স্পেস সহজ ফরম্যাটে দেখুন
uname -a
Show kernel & system architecture info
কার্নেল ও সিস্টেম আর্কিটেকচার তথ্য দেখুন
ps aux
List all active processes
সব চলমান প্রসেসের তালিকা দেখুন
🌐
curl / wget
Download files and make HTTP requests from terminal
টার্মিনাল থেকে ফাইল ডাউনলোড ও HTTP রিকোয়েস্ট করুন
NETWORK
curl https://example.com
Fetch and print a webpage's HTML
ওয়েবপেজের HTML প্রিন্ট করুন
wget https://file.com/file.zip
Download a file from internet
ইন্টারনেট থেকে ফাইল ডাউনলোড করুন
curl -X POST -d 'data' url
Send POST request with data to a URL
URL-এ POST রিকোয়েস্ট ডেটা সহ পাঠান
🔍
Nikto
Web server vulnerability & misconfiguration scanner
ওয়েব সার্ভার ভালনারেবিলিটি স্ক্যানার
SECURITY
INSTALL
ইন্সটল
pkg install nikto
COMMANDS
কমান্ড
nikto -h http://target.com
Scan a web server for vulnerabilities
ওয়েব সার্ভার ভালনারেবিলিটির জন্য স্ক্যান করুন
nikto -h target -o result.txt
Scan and save results to a file
স্ক্যান করে ফাইলে রেজাল্ট সেভ করুন
🐍
Python3
Run Python scripts, automation & exploitation tools
Python স্ক্রিপ্ট চালান, অটোমেশন ও এক্সপ্লয়েট টুলস
SCRIPT
INSTALL
ইন্সটল
pkg install python
COMMANDS
কমান্ড
python3 script.py
Run a Python script
Python স্ক্রিপ্ট চালান
pip install requests
Install a Python library using pip
pip দিয়ে Python লাইব্রেরি ইন্সটল করুন
python3 -m http.server 8080
Start a quick HTTP server on port 8080
পোর্ট 8080-এ দ্রুত HTTP সার্ভার চালু করুন

Git
Clone GitHub repos & manage version control
GitHub রিপো ক্লোন করুন ও ভার্শন কন্ট্রোল ম্যানেজ করুন
GIT
INSTALL
ইন্সটল

COMMANDS
কমান্ড
git clone https://github.com/user/repo
Clone a GitHub repository locally
GitHub রিপো লোকালি ক্লোন করুন
git pull
Pull latest updates from remote repo
রিমোট রিপো থেকে লেটেস্ট আপডেট নিন

Metasploit
Penetration testing framework — find & exploit vulnerabilities
পেনিট্রেশন টেস্টিং ফ্রেমওয়ার্ক — ভালনারেবিলিটি খুঁজুন ও এক্সপ্লয়েট করুন

Netcat (nc)
